# Badminton Tournament Manager

A flexible badminton tournament management web app built with Next.js, TypeScript, Tailwind CSS, shadcn/ui, Prisma, and PostgreSQL.

## Current status

Project foundation is being set up.

## Stack

- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui
- Prisma
- PostgreSQL

## Goal

Support flexible badminton tournament formats, long-term player records, and admin-friendly tournament operations.
